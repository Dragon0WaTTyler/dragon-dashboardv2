# Product

## Register

product

## Users

Dragon is a private, single-user workspace for its owner. It is used for short daily
check-ins and focused sessions across movies, video, reading, books, chess, learning,
and personal history.

## Product Purpose

Dragon keeps personal media and learning data local, makes the next useful action
obvious, and preserves enough source and freshness context for the owner to trust what
they see. External work happens only after a deliberate action; ordinary page rendering
remains fast and local.

## Brand Personality

Focused, private, refined. The product should feel decisive and calm while retaining a
distinctive premium character.

## Anti-references

- Generic Bootstrap or SaaS dashboards.
- Streaming-service clones and full-screen cinematic hero layouts.
- The legacy Cinema Prive monolith, navigation, and page composition.
- Crowded cards, oversized headings, decorative controls, and hidden network behavior.

## Design Principles

1. Put the current task before decoration.
2. Keep local data and system state legible and trustworthy.
3. Reveal depth progressively through deliberate user actions.
4. Treat Arabic and left-to-right content as equal first-class layouts.
5. Preserve one coherent interaction vocabulary across every module.

## Accessibility & Inclusion

Target WCAG AA contrast, semantic landmarks, visible keyboard focus, labelled controls,
44-pixel minimum touch targets, useful image alternatives, reduced-motion support, and
responsive layouts without page-level horizontal overflow. Bidirectional text must keep
natural reading direction and predictable control placement.
