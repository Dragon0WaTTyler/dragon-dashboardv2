## Architecture and safety

Each domain uses Blueprint/routes, services, repositories where applicable,
models, templates, and API projections. Page rendering is local-only. Domain
mutations are CSRF protected and recorded in local history. External operations
are explicit and return a safe report when disabled or unconfigured.

No legacy frontend assets, CSS, JavaScript, templates, navigation, identity, or
page layouts were copied. The `frontend-ui-engineering` skill influenced the
component composition, URL-owned filters, accessibility states, responsive
behavior, and avoidance of nested dashboard-card layouts.

## Deliberate gates

- No legacy personal records were copied or imported.
- No external provider credentials are present.
- Provider-specific network adapters remain disabled/unconfigured; local
  functionality and safe fallbacks remain available.
- Playback source approval never starts a media/torrent process.
